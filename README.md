# STM32-M4-Low-Power-Lighting-Detector-Based-On-OpenHarmony
The repository just for my competition for OpenHarmony.But if you like, you can use them.

Goals
1. The device can Connect to the nleCloud platfrom. url: http://www.nlecloud.com/
2. It can be three LowPowerModes and controlled by MatrixButton 1, 5 and 9 ;three LowPowerModes means different screen dispaly times: forever 5s and 10s.
3. The OLED screen must display : Netstate, CloudIp, Light, and LowPowerMode.
